EOB Agentic Solution

Interactive Architecture & Technical Documentation

1. Overview

The EOB Agentic Solution is a state-of-the-art FastAPI service designed to extract high-accuracy structured data from complex medicinal insurance claims (VSP, EyeMed, etc.).

🛡️ Self-Healing

Automatic Auditor-Critic loops ensure 100% data integrity without human intervention.

🧠 Deep Context

Remembers document history across batches to resolve hierarchical claim dependencies.

⚡ Massive Scale

Elastic chunking allows processing of 100+ page PDFs with consistent sub-second page throughput.

2. The Journey of a Request

01 Ingestion & Synthesis

The API receives the PDF. FormatLoader dynamically binds Pydantic schemas, and PDFProcessor prepares structural grids using Azure AI.

02 Multimodal Analysis

The system fuses OCR text with high-res visual snapshots. This allows the agents to "see" the document layout and table borders perfectly.

03 The Agentic Crucible

LangGraph orchestrates the feedback loop: Extractor → Auditor → Critic. Missing values are detected and re-extracted automatically.

04 Final Materialization

Results are aggregated, totals are calculated, and the final validated JSON is delivered to the client and saved locally.

🤖 The Agent Squad

Orchestrator extraction_graph.py

The "Conductor". Uses LangGraph to route data between agents in a cyclic feedback loop, managing the state and retry limits.

Extractor gemini_agent.py

The "Muscle". Multi-modal extraction using Gemini 1.5 Pro. Handles continuation context and learning memory.

Auditor auditor_agent.py

The "Inspector". Acts as a fresh eye, comparing the extracted result against the source PDF to find discrepancies.

Critic critic_agent.py

The "Coach". Converts Auditor issues into clear, numbered improvement instructions for the next loop.

Manager agent_factory.py

The "HR". Reads config to hire (instantiate) either Gemini or Claude for the various agent roles.

Generator format_generator.py

The "Onboarder". Automatically analyzes new PDF types and generates the Python Pydantic format code.
